1. Open app screen watch

2. Press the key combination Ctrl + Alt + Shift + S to start.
   Program will run hide
		Pressed
	and result....
3. After each 2s, program will write screen image on file
   with format name file "Screen yymmdd hhmmss.bmp"
   All file image save in folder "Image_ScreenWatch"

4. Press the key combination Ctrl + Alt + Shift + S again to end
   Program will appear become

5. Press File=> exit or press 'X' to exit
